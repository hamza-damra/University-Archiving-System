How to Test

  1. Start the application: .\mvnw.cmd spring-boot:run
  2. Login as an HOD user
  3. Navigate to the "Reports" tab in the sidebar
  4. Select an Academic Year and Semester from the context bar
  5. Apply filters (Course, Professor, Document Type, Status, Date Range)
  6. Click "Generate Report" to view results
  7. Use pagination controls to navigate through results
  8. Click "Export PDF" to download the report
  9. Click "Clear Filters" to reset all filters
