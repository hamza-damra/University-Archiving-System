Collapsible Sidebar Test

Test Instructions

  1. Click the "Collapse" button at the bottom of the sidebar
  2. Verify the sidebar collapses to 64px width with only icons visible
  3. Verify the collapse button icon rotates 180 degrees
  4. Hover over nav items to see tooltips when collapsed
  5. Click the collapse button again to expand
  6. Verify the sidebar expands back to 260px with labels visible
  7. Reload the page and verify the collapsed state persists
  8. Check that the transition is smooth (0.3s ease)

Expected Behavior:

  • Smooth 0.3s ease transition animation
  • Tooltips appear on hover when collapsed
  • State persists in localStorage
  • Icons remain centered when collapsed
  • All labels hidden when collapsed
