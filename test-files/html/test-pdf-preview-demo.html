PDF Renderer Demo

This demo shows the PDF renderer with navigation controls. The renderer uses the browser's native PDF viewer with custom navigation.

Features Implemented:

  • PDF rendering via iframe with browser's native viewer
  • Page navigation controls (Previous, Next, Jump to page)
  • Current page and total pages display
  • Page number input with validation
  • Boundary condition handling (page clamping)
  • Keyboard support (Enter key in page input)
